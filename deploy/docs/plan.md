# 前端部署学习计划 (15天)

## 计划概述

本计划将帮助你系统掌握前端项目从开发到生产部署的全流程，包括静态资源部署、服务器配置、CI/CD、容器化、云服务等核心知识点。

## 学习目标

基于以下5个核心目标制定学习计划：

1. **独立部署个人应用到域名**：掌握完整的前端部署流程，能够将个人项目部署到自定义域名
2. **分析和优化公司部署方案**：深入理解常见部署方案，具备分析现有方案优缺点并提出改进建议的能力
3. **Docker 优化部署**：掌握 Docker 容器化部署，包括镜像优化、构建加速等最佳实践
4. **对象存储时空优化**：精通对象存储服务（OSS/COS/S3），优化存储成本和访问性能
5. **CI/CD Pipeline 优化**：掌握 CI/CD 流程优化技巧，缩短部署时间，支持敏捷开发和快速迭代

## 核心技能目标

- 掌握静态资源部署和优化
- 熟练使用 Nginx 进行反向代理和负载均衡
- 理解 CI/CD 流程并实践自动化部署和优化
- 掌握 Docker 容器化部署和性能优化
- 精通云服务部署和对象存储优化
- 掌握性能监控和故障排查

## 学习计划

### 第一阶段：基础部署（Day 1-4）

#### Day 1: 静态资源部署基础

- **学习内容**
  - 静态资源打包和构建（webpack/vite 生产配置）
  - 资源压缩和优化（Gzip、Brotli）
  - 缓存策略（浏览器缓存、CDN缓存策略）
  - 资源路径配置（绝对路径 vs 相对路径）
  - 代码分割和懒加载优化
- **实践任务**
  - 配置 webpack/vite 生产构建
  - 实践资源压缩和代码分割
  - 配置缓存策略（Cache-Control、ETag）
  - 对比优化前后的资源大小和加载时间

#### Day 2: HTTP 服务器基础

- **学习内容**
  - HTTP 协议基础（状态码、请求头、响应头）
  - 静态文件服务器（Express、Koa）
  - MIME 类型配置
  - CORS 跨域处理
  - 安全头配置（XSS、CSRF防护）
- **实践任务**
  - 使用 Node.js 搭建静态文件服务器
  - 配置 CORS 和 MIME 类型
  - 实现简单的路由和中间件
  - 配置安全响应头

#### Day 3: Nginx 基础配置

- **学习内容**
  - Nginx 安装和启动（macOS、Linux）
  - 基本配置文件结构（http、server、location）
  - 静态文件服务配置
  - 日志配置和查看（access.log、error.log）
  - 基础性能优化（worker_processes、keepalive）
- **实践任务**
  - 在本地安装 Nginx
  - 配置简单的静态文件服务
  - 实践日志分析（日志格式、日志切割）
  - 测试静态文件访问性能

#### Day 4: Nginx 进阶配置

- **学习内容**
  - 反向代理配置（proxy_pass）
  - 负载均衡算法（轮询、权重、IP hash、least_conn）
  - 动静分离配置
  - URL 重写规则（rewrite、return）
  - 请求头和响应头处理
- **实践任务**
  - 配置反向代理到后端 API
  - 实践负载均衡配置（多后端服务）
  - 实现 URL 重写规则（SPA 路由支持）
  - 配置请求头转发

### 第二阶段：服务器部署（Day 5-8）

#### Day 5: Linux 服务器基础

- **学习内容**
  - Linux 常用命令（文件操作、进程管理、网络配置）
  - 文件权限管理（chmod、chown）
  - 进程管理（pm2、systemd）
  - 防火墙配置（iptables、ufw）
  - 用户和权限管理（sudo、SSH 密钥）
- **实践任务**
  - 练习 Linux 基础命令
  - 配置进程守护（pm2 start、pm2 ecosystem）
  - 配置防火墙规则（开放端口）
  - 配置 SSH 密钥登录

#### Day 6: 服务器环境搭建和域名配置

- **学习内容**
  - Node.js 环境安装（nvm 管理多版本）
  - SSL 证书申请和配置（Let's Encrypt、acme.sh）
  - 域名解析和 DNS 配置（A记录、CNAME、泛域名）
  - 域名备案流程（国内服务器）
  - 域名解析检查工具（dig、nslookup）
  - 证书自动续期配置
- **实践任务**
  - 在服务器上搭建 Node.js 环境
  - 申请并配置 SSL 证书（支持自动续期）
  - 配置域名解析，实现个人项目通过域名访问
  - 验证 HTTPS 配置和证书有效性
- **核心目标**：完成个人应用部署到域名的完整流程

#### Day 7: 项目部署实战和方案对比

- **学习内容**
  - 项目上传和部署流程（Git、FTP、SCP、rsync）
  - 环境变量配置（.env 文件管理、环境隔离）
  - 多环境部署策略（开发、测试、生产）
  - **常见部署方案对比分析**：
    - 传统服务器部署（Nginx + 静态文件）
    - Serverless 部署（Vercel、Netlify）
    - 容器化部署（Docker）
    - 对象存储 + CDN 部署
  - 部署方案优缺点分析框架
  - 部署方案选型指南
- **实践任务**
  - 部署一个完整的 Vue/React 项目到个人域名
  - 配置生产环境变量和多环境切换
  - 分析并对比至少2种不同的部署方案
  - 编写部署方案对比文档
  - **（可选）调研当前公司部署方案，识别优化点**
- **核心目标**：能够独立部署个人应用，并具备分析公司部署方案的能力

#### Day 8: Nginx 高级配置

- **学习内容**
  - 限流配置（rate limiting、连接限制）
  - 安全配置（防盗链、防爬虫、WAF规则）
  - 压缩配置优化（Gzip、Brotli）
  - 多站点配置（server blocks、虚拟主机）
  - 日志分析和监控
- **实践任务**
  - 配置接口限流（防止API滥用）
  - 实现防盗链和安全头配置
  - 配置多站点 Nginx
  - 配置日志切割和归档
