# 项目运行
```
$ git clone git@github.com:kankan-web/web-project-example.git

# 进入到示例文件
$ cd web-project-example

# 在根目录直接安装包
$ npm i

# 进入对应的目录
$ cd webpack/cache

# 执行对应的文件，查看文件中的示例及注释
$ node index.js
```
# 目录结构
- webpack:关于一些简单的webpack示例
- node：关于一些简单的node示例
